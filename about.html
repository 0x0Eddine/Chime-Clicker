Background

Chime Clicker is a programming side project of mine about 9 months in the making. After getting hooked on incremental games a couple years back, I was inspired to build my own. As an avid League of Legends player, when Bard’s chime collecting mini-game was revealed, I knew I the perfect theme for it. Chime Clicker has been a fun (and productive) alternative to grinding solo queue and spamming ARAMs. I’ve learned a lot so far and look forward to maintaining the game and continuing to add new features.


Tools and Technologies

HTML, SASS, Javascript, AngularJS (modules: order-object-by), jQuery (plugins: hotkeys, simplemodal, tooltipster)

Bugs

When reporting bugs, try to include [Bug] in the subject line/title of any email/reddit post along with any information you think might be relevant, most notably which browser you are using (I mostly tested using Chrome, so I expect a lot of formatting/layout issues across other browsers and on mobile). Screenshots and steps to reproduce will go a long way in helping me fix any issues you guys encounter. Thanks in advance!


A list of known issues and planned features can be found on the Issues page.
